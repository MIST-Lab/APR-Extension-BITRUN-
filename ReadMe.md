##### 产品方案

###### AutoCorectorIDE

从代码提供者的代码中识别和保存正确的代码，并对其给予等价值的奖励。
将正确的代码发送给需要修改者，并收取相应价值的money。
通过这种方式，可以自动利用相应的综合知识和对应劳动力来改进每个代码。

### 技术难点说明
* Name Translation:
难点之一是将值的命名从代码提供者的名称空间转换为需要修改者的名称空间。
